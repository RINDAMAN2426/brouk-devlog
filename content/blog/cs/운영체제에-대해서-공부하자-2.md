# 다중 처리 시스템

성능 향상, 다른 처리기가 고장이 나더라도 시스템은 정상적으로 동작

## 대칭적 다중 처리 시스템

- 각 CPU가 하나의 운영체제 아래 작동.
- CPU가 데이터를 공유하는 구조로 운용

## 비대칭적 다중 처리 시스템

- 각 처리기마다 특정한 작업
- 하나의 주요 CPU가 제어, 다른 CPU는 정의된 작업이나 명령을 수행

# 분산 처리 시스템

- 네트워크를 이용해 동시에 작업 처리
- CPU들이 메모리 공유를 하지 않음
  > - 클라이언트 & 서버 시스템
  > - P2P 시스템

# 클러스터 시스템

- 물리적으로 많은 CPU를 한 곳에 모아서 특정한 작업을 수행
- 기상청의 시스템
- LAN으로 서로 연결

> - 대칭적 클러스터 시스템
> - 비대칭적 클러스터 시스템
